Misc. Notes

  • SSH: Secure Shell, a network communication protocol that enables two computers to communicate, and share data.
  • MVP: Minimum Viable Product, usually the first iteration of a product without extra features.
  • Version Control Systems: This allows us to track and manage changes to our codebase. If we need to make changes, we can without losing other versions of our code.
  • SHA or Hash: Allows you to keep a record of the specific changes committed. Example; Hash looks like a giant string of numbers and letters, developers often use onlu the first seven numbers/letters.
  • Graphical User Interface: (GUI) A way for users to communicate and interact with the computer visually.
  • Intergrated Development Environment: (IDE) A software for developers that combines a myraid of features needed to do their work in a simple interface.
  • Pull Request: (PR) Is used to merge a comparison branch into the base branch.
  • Default Branch: Is called main, it should be the final most updated version of the application.
  • Pair Programming: When two developers team up on one computer.
  • Version Control: The practice of tracking and managing changes to software code.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.
  • The href attribute is using a relative path. This means the location is derived from the current location. To locate the style sheet from the current location, we must look in the assets folder, hence assets/style.css.
  • An Array is a single varible that is used to hold a group of data. Typically used to hold data that is related in some way.
  • A function is a set of instructions that tells the computer how to perform a certain task

HTML

    • Elements on a page are used to add content and also used to divide the content on the page. Some examples are: Text, Images, Header, and Lists.
    • The head element contains information about the webpage.
    • The body element represents the visible content shown to the user.
    • The meta element contain info about the page that is used by the browser.
    • The section element are elements that define a section in a document that contains thematic content.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript makes a webpage able to have interactions with users
  • A varible is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
